# To-Do List Application 

Prosta i funkcjonalna aplikacja do zarządzania codziennymi zadaniami, stworzona jako projekt zaliczeniowy do portfolio na praktyki zawodowe.

## Funkcje
- **Dodawanie zadań:** Szybkie wprowadzanie nowych punktów do listy.
- **Interaktywność:** Możliwość oznaczania zadań jako wykonane (przekreślenie).
- **Usuwanie:** Opcja trwałego usunięcia zadania z listy.
- **Persistence (LocalStorage):** Twoje zadania są bezpieczne – nie znikną po odświeżeniu przeglądarki!
- **Responsywność:** Aplikacja działa poprawnie na komputerach i urządzeniach mobilnych.

## Technologie
- **HTML5** – semantyczna struktura strony.
- **CSS3** – nowoczesny layout (Flexbox) i stylizacja.
- **JavaScript (Vanilla JS)** – logika aplikacji i obsługa LocalStorage.
